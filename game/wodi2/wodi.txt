
**游戏名称：** 谁是卧底

**游戏类型：** 语言描述、推理

**适合人数：** 6人

**游戏目标：**
*   **平民的目标：** 通过描述和投票，找出隐藏在你们中间的卧底并将其淘汰。
*   **卧底的目标：** 隐藏自己的身份，模仿平民的描述，避免被投票淘汰，并争取在游戏结束时存活下来（或者在人数劣势时淘汰所有平民）。
*   **白板的目标（若有）：** 通过分析，推断出真正的词语是什么，并努力存活到最后。

**所需道具：**
1.  **词语卡片/纸条：** 准备两组（或多组）含义非常相近但又有微妙区别的词语（通常是名词）。每组词语包含：
    *   **1张“卧底词”卡片：** 给卧底玩家。
    *   **若干张“平民词”卡片：** 数量等于平民玩家数（总人数-1-白板数），所有平民拿到的是同一个词语。
    *   **（可选）1张“白板”卡片：** 给白板玩家（空白或写着“白板”）。
2.  **笔（用于写词语）：** 如果现场准备词语。
3.  **主持人：** 负责分发词语、引导流程、判断描述是否违规、宣布结果。主持人通常不参与游戏本身（也可以由玩家兼任，但需公平）。
4.  **（可选）计时器：** 用于限制每位玩家的描述时间（通常30秒-1分钟）。

**核心概念：**
*   **平民词：** 大部分玩家拿到的词语，含义相同。
*   **卧底词：** 卧底玩家拿到的词语，与平民词高度相似但有明显区别。
*   **白板：** （可选角色）没有拿到任何词语的玩家，需要全程通过他人的描述来猜测词语是什么。
*   **描述：** 玩家用一句话（或一个词/短语）描述自己拿到的词语，不能直接说出词语本身或其中的字。
*   **投票：** 每轮描述结束后，玩家投票选出他们认为最可能是卧底的人。

**详细游戏流程：**

1.  **准备阶段：**
    *   确定参与玩家和主持人。
    *   主持人根据玩家人数准备词语卡片：
        *   例如，6人游戏：准备1张“卧底词”（词A），4张“平民词”（词B），1张“白板”（可选）。总词卡数=玩家数。
        *   主持人私下洗混所有词卡（包括卧底词、平民词、白板卡），确保只有主持人知道每张卡片的归属。
    *   主持人将卡片**背面向上**分发给每位玩家。**玩家在游戏开始前，只能看自己的卡片！** 不能互相交流卡片内容。

2.  **第一轮描述：**
    *   主持人指定或随机决定一位玩家开始进行描述。
    *   **描述规则：**
        *   玩家**只能用一句话（或一个词/短语）** 来描述自己拿到的词语。
        *   **绝对禁止**直接说出词语本身或词语中包含的任何字！
        *   **描述必须与自己的词语相关，且尽量准确。**
        *   平民描述的是平民词（词B），卧底描述的是卧底词（词A），白板则需要根据之前的描述**假装自己也有一个词语**来进行模糊描述。
        *   描述内容可以是：物品的外观、功能、性质、类别、使用场景、相关联想等。例如，如果词是“包子”，可以说“是一种面食”、“有馅”、“早餐常吃”、“蒸出来的”。
        *   描述应**避免过于宽泛**（如“是一种东西”）或**过于具体暴露**（如“皮薄馅大十八个褶” - 这几乎直接指向包子）。
        *   主持人需要判断描述是否违规（如直接说出字词），若违规需要求重说或给予警告。
    *   描述按**顺时针或逆时针**顺序依次进行。每位玩家描述时，其他玩家**认真倾听并思考**。
    *   （可选）使用计时器限制描述时间，增加紧张感。

3.  **第一轮投票：**
    *   所有玩家描述完毕后，进入投票环节。
    *   **投票规则：**
        *   玩家同时（或依次）**用手指向**自己认为最可能是卧底的人。
        *   **不能弃权**（除非规则特别说明）。
        *   主持人统计票数。
        *   **得票最多的玩家被淘汰出局。**
        *   如果出现**平票**（两人或多人得票相同且最高），则平票的玩家进行**一轮限时的自我辩护**（简短说明自己不是卧底的理由），然后**所有存活玩家在平票者中再次投票**。若再次平票，通常无人被淘汰（或根据提前约定的规则处理）。
    *   被淘汰的玩家**亮出自己的词语卡片**，揭示其身份（是平民、卧底还是白板）。**游戏继续。**

4.  **后续轮次：**
    *   重复步骤2（描述）和步骤3（投票）。
    *   **描述顺序：** 通常从被淘汰玩家的下一位开始描述。
    *   **关键变化：**
        *   随着玩家的淘汰，信息逐渐增多。平民需要根据淘汰者的身份（是平民则说明大家投错了，卧底还活着；是卧底则说明危险解除）和所有人的描述，更精准地推理。
        *   卧底需要根据平民的描述，努力推断出平民词是什么，并调整自己的描述进行模仿和伪装。
        *   白板需要更加仔细地倾听和分析，尝试拼凑出词语信息，同时也要模仿他人，避免被怀疑。
        *   投票变得更加关键和困难，因为人数减少，每一票都至关重要。

5.  **游戏结束与胜负判定：**
    *   游戏在以下情况之一时结束：
        *   **卧底被淘汰：** **平民阵营获胜！**
        *   **平民/白板被淘汰到只剩1人（或约定的少数），而卧底仍然存活：** **卧底获胜！** (例如，6人局：剩1平民+1卧底，卧底赢；剩1白板+1卧底，卧底赢)。
        *   **（若有白板）白板存活到最后（无论卧底是否被淘汰），且能在游戏结束时准确说出平民词是什么：** **白板单独获胜！** (这是白板获胜的唯一方式)。如果白板猜错词语，则即使存活到最后也不算赢。
    *   主持人宣布游戏结果，所有玩家亮出词语卡片确认。

**词语选择技巧（主持人关键）：**

*   **核心原则：** 平民词和卧底词必须**高度相似但有明确区别**。
*   **好的词语组合示例：**
    *   包子 / 饺子
    *   牛奶 / 豆浆
    *   电梯 / 楼梯
    *   博客 / 微博
    *   哈密瓜 / 香瓜
    *   薰衣草 / 迷迭香
    *   汉堡 / 三明治
    *   吉他 / 尤克里里
    *   冰箱 / 空调
    *   西红柿 / 圣女果
    *   唇膏 / 口红
*   **避免的词语：**
    *   区别太大（如：汽车 / 苹果） - 游戏瞬间结束，无趣。
    *   区别过于细微或需要专业知识（如：不同品种的玫瑰花） - 难以描述和分辨。
    *   包含生僻字或过于冷门的事物。
    *   过于宽泛或抽象的词语（如：爱 / 恨）。
*   **难度调整：** 词语的相似度决定了游戏难度。相似度越高，卧底越难被发现，游戏越烧脑；相似度稍低，游戏节奏更快。

**玩家技巧：**

*   **平民：**
    *   **描述准确但留有余地：** 既要清晰传达词语信息，帮助同伴识别，又不能说得太直白让卧底轻易猜出平民词。
    *   **仔细倾听所有人描述：** 特别注意谁的说法和自己有微妙的不同，或者谁的说法模糊不清、模棱两可。
    *   **分析投票结果：** 谁被投出去了？他是平民还是卧底？这对判断剩下的人很重要。
    *   **不要轻易跟风：** 要有自己的分析和判断。
*   **卧底：**
    *   **第一轮模仿是关键：** 首轮描述尽量模糊、中性，贴合平民可能描述的范畴，避免暴露差异点。认真听平民的描述，努力猜出平民词。
    *   **后续调整：** 一旦猜出或大致确定平民词，后续描述要努力模仿平民的说法，融入集体。
    *   **利用矛盾：** 可以尝试引导大家怀疑描述得过于清晰或过于模糊的玩家。
    *   **保持镇定：** 即使被怀疑，也要冷静描述，避免慌乱出错。
*   **白板：**
    *   **超级倾听者：** 你的主要任务是收集信息。认真听每个人的每一句话，寻找共同点和差异点。
    *   **模糊描述大师：** 你的描述要听起来像知道词语，但又不能暴露自己不知道。可以用“大家说的都挺对”、“这个物品很常见”、“它的特征很明显”这类非常模糊的话。也可以尝试模仿前一位玩家的描述角度。
    *   **观察投票：** 投票结果能告诉你谁被怀疑了，这有助于推断词语。
    *   **后期发力：** 随着信息增多，你的描述可以稍微大胆一点，接近你猜测的词语，但要非常小心。
*   **所有人：**
    *   **注意表情和语气：** 虽然规则不禁止，但过于紧张或刻意掩饰反而容易引起怀疑。
    *   **享受推理和表演的过程！**

**主持人注意事项：**

*   **公平公正：** 严格判断描述是否违规，确保投票过程清晰。
*   **清晰引导：** 明确告知当前环节（描述中/投票中），维持游戏秩序。
*   **控制节奏：** 适当推进流程，避免讨论时间过长导致冷场或混乱。
*   **保密：** 绝不提前泄露词语分配。
*   **灵活处理：** 对规则争议或特殊情况（如持续平票）有预案或能做出合理裁决。
*   **营造氛围：** 让游戏过程轻松愉快，充满悬念。

**游戏变体：**

*   **双卧底/多卧底：** 在人数较多（如8人）时，可以设置2个卧底（拿同一个卧底词）甚至1个卧底+1个白板。增加游戏复杂度和卧底方的力量。
*   **词语难度分级：** 准备不同难度的词语组，玩家可以自行选择。
*   **描述方式限制：** 例如，只能用形容词描述，或者只能用动作描述，增加挑战性。
*   **不亮身份淘汰：** 被淘汰者不亮牌，直到游戏结束才揭晓所有身份，悬念保留更久（但对玩家推理能力要求更高）。
