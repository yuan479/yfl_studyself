# 《火柴人跑酷》游戏设计方案

## 游戏概述
《火柴人跑酷》是一款横版跑酷游戏，玩家控制火柴人角色在充满挑战的环境中奔跑、跳跃、滑行和战斗，目标是尽可能跑得远并收集资源。游戏融合了动作、平台跳跃和轻度RPG元素，提供紧张刺激的游戏体验。

**功能描述：**
- 顶部状态栏：
  - 角色头像和生命值(HP)显示
  - 能量条（用于特殊技能）
  - 当前奔跑距离
  - 金币数量
- 游戏主区域：
  - 横向卷轴场景，随角色移动自动滚动
  - 火柴人角色位于屏幕左侧1/3处
  - 动态生成的平台、障碍物和敌人
- 底部控制栏：
  - 四个触控按钮：跳跃(空格)、攻击（f）、滑行（a）、技能（j）
  - 按钮有冷却时间视觉反馈

## 核心游戏机制

### 1. 跑酷移动系统
- **自动前进**：火柴人持续向右奔跑
- **环境互动**：
  - 跳跃：越过沟壑和低矮障碍
  - 滑行：通过狭窄通道
  - 墙面奔跑：在特定墙面上水平移动
  - 滑索：快速穿越长距离
- **移动技巧**：
  - 连跳：连续跳跃获得额外高度
  - 冲刺：短暂加速消耗能量
  - 翻滚：躲避攻击时减少伤害

### 2. 战斗系统
- **近战攻击**：拳击和踢腿，可形成连击
- **远程攻击**：消耗能量发射飞镖或火球
- **特殊技能**：
  - 时间减缓：短暂减慢游戏速度
  - 地震波：击倒附近敌人
  - 火焰冲刺：点燃路径上的敌人
- **敌人类型**：
  - 僵尸：低速高生命
  - 忍者：高速低生命
  - 机器人：远程攻击
  - BOSS：关卡终点强力敌人

### 3. 道具收集系统
| 道具类型 | 效果 | 外观 |
|---------|------|------|
| 金币 | 游戏货币 | 金色硬币 |
| 生命药水 | 恢复HP | 红色药瓶 |
| 能量饮料 | 恢复能量 | 蓝色罐子 |
| 磁铁 | 吸引附近道具 | U形磁铁图标 |
| 护盾 | 短暂无敌 | 透明护罩 |

### 4. 角色成长系统
- **等级提升**：通过奔跑距离和击败敌人获得经验
- **属性成长**：
  - 生命值：增加容错率
  - 速度：提升基础移动速度
  - 跳跃：增加跳跃高度
  - 攻击：增强伤害输出

## 关卡设计

### 2. 动态难度系统
- **自适应速度**：根据玩家表现调整游戏速度
- **智能障碍生成**：基于玩家位置动态创建挑战
- **BOSS战机制**：
  - 阶段变化：生命值降至不同阶段改变攻击模式
  - 弱点系统：特定部位受到额外伤害
  - 环境互动：利用场景元素对抗BOSS

## 游戏模式

### 1. 经典模式
- 无尽跑酷，距离为主要目标
- 难度随时间逐渐增加


### 1. 物理引擎
- **碰撞检测**：像素级精确碰撞
- **运动系统**：
  - 重力模拟
  - 惯性计算
  - 摩擦力模拟

### 2. 视觉效果
- **火柴人动画**：
  - 流畅的骨骼动画
  - 动作融合技术
- **粒子系统**：
  - 奔跑尘土
  - 攻击特效
  - 道具收集效果


## 辅助功能

### 1. 难度选项
- 简单模式：减少障碍物
- 普通模式：标准体验
- 困难模式：额外挑战

### 2. 视觉辅助
- 高对比度模式
- 色盲友好配色
- 界面缩放选项
